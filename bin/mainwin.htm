Создание окна

2. Запись главной функции MainWin и регистрация окна приложения

Отправной точкой всех приложений Windows является функция WinMain(), в которой происходит создание окна и обработка сообщений. Сообщение - это центральное понятие в среде Windows. Система посылает сообщения приложениям, а те, в свою очередь, должны на них правильно отреагировать. Получателями сообщений являются функции окон приложений, на программирование которых и уходит большая часть времени.
Сейчас перед вами стоят задачи:
- Написать функцию MainWin();
- Поместить в ней код регистрации окна в системе;

Задание 6
Для начала скопируйте архив MRCD9.zip , в папку, где располагается ваш проект-файл (Direct3D.dsw) и разархивируйте его в эту же папку.

Но простая разархивация вам не поможет. Перед использованием данных файлов нужно добавить их в проект.

Задание 8
1) Откройте ваш проект (если он закрыт). Проекты имеют расширение *.dsw .
2) Перейдите на файл D3DApplication.cpp. Для этого кликните по иконке Window List на панели инструментов и в открывшемся окне выберите D3DApplication.cpp . Должен открыться белый лист.
3) Напишите на нем код подключения библиотек для работы с классами MRCD9 и ускоренной регистрации вашего окна в системе:

#include "MRCD9.H"

Задание 9
После подключения библиотек впишите пустую функцию MainWin() как показано ниже:

int WINAPI WinMain
(
HINSTANCE hinstance,
HINSTANCE hprevinstance,
LPSTR lpcmdline,
int ncmdshow
)
{

};

Для регистрации нашего будущего окна в системе воспользуйтесь функцией InitMainForm.
Вот её прототип:

HWND InitMainForm
(
HINSTANCE hinstance,
WNDPROC lpfnWndProc,
LPCTSTR namef,
int width,
int heihgt
)

Функция InitMainForm имеет следующие параметры:
- hinstance - метка окна (примерно так же как паспорт у человека);
- lpfnWndProc - это указатель на функцию обратного вызова (Про неё поговорим чуть позже) .
- namef - имя окна;
- width - ширина создаваемого окна;
- heihgt - высота;
Данная функция возвратит дескриптор, который в дальнейшем будет использоваться для работы с сообщениями windows.

Задание 10
Внутри операторных скобок функции WinMain() запишите следующее:

HWND hwnd = InitMainForm(hinstance, MainWinProc, "D3D", 800, 600);

И так главную функцию записали, окно зарегистрировали. Теперь осталось создать обработчик событий. Для этого жмите "далее".
