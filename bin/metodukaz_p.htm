Рекомендации для преподавателей:

Перед началом обучения для определения соответствия уровня подготовки обучаемых минимальным требованиям рекомендуется каждому обучаемому пройти входной тестовый контроль. Переход к обучению возможен только в том случае, если в результате контроля обучаемый набрал более трех баллов из пяти возможных. Если обучаемым были допущены ошибки, то в конце контроля при показе оценки за него будет выведен перечень тем для повторения.

После завершения обучения по данному электронному пособию рекомендуется каждому обучаемому выполнить итоговый тестовый контроль. Материал пособия считается усвоенный удовлетворительно, если обучаемый набрал в результате контроля более трех баллов из пяти возможных. Иначе следует повторить темы, которые будут выведены на экран вместе с оценкой за контроль.

После положительного выполнения входного контроля обучаемые должны перейти непосредственно к выполнению лабораторных работ. Начать выполнять лабораторные работы возможно, перейдя по ссылке «Обучение» на главной странице .

В ходе выполнения лабораторных работ обучаемым будут встречаться завуалированные задания для самостоятельной работы, с помощью которых преподаватель может контролировать процесс обучения. К таким заданиям относятся:
1. задание 26 в лабораторной работе «Создание нейросети»,
2. задание 7 в лабораторной работе «ВМА-файл»,
3. задание 14 в лабораторной работе «Экспорт модели»,
4. задание 41 в лабораторной работе «Внедрение нейросети».

Все выше описанные задания творческого характера, которые выполняются индивидуально. Верно, выполненным считается то задание, которое реализует цель, изложенную в тексте задании, не зависимо от способа реализации.

Пример реализации заданий для самостоятельной работы находятся в папке «Для преподавателя».

В конце каждой лабораторной работы находиться блок «Вопросы для самоконтроля», которые помогают контролировать ход педагогического процесса. Вопросы охватывают все учебные элементы, изучаемые в лабораторной работе. Рекомендуется после каждой лабораторной работы опрашивать по данным вопросам обучаемых, для того чтобы выявить слабо усвоенные учебные элементы.

Лабораторные работы следует изучать в следующем порядке:
1. «Создание окна»,
2. «Создание нейросети»,
3. «Запуск Direct3D»,
4. «Визуализация»,
5. «Анимация»,
6. «Камера»,
7. «ВМА-файл»,
8. «Экспорт модели»,
9. «Внедрение нейросети».

Эталоны тестовых заданий для входного и итогового тестовых контролей находятся в папке «Для преподавателей».
