1. Создание объекта нейросети

Попытки моделирования работы мозга с помощью построения моделей нейронных сетей (нейрокибернетика) восходят ко времени появления первых ЭВМ и не были напрямую связаны с их появлением. Первые модели нейроподобных сетей были физическими и основывались на электронных моделях нейронов. Начало этому моделированию положила идея формального нейрона Мак-Каллока и Питтса (1943 г.), которая была воспринята Розенблаттом и развита им в рамках построенных им и его учениками многочисленных моделейперцептронов (систем для распознавания образов) - одного из классов современных нейронных сетей. По существу, Розенблатта можно считать отцом нейрокибернетики или нейрокибернетического направления в Искусственном Интеллекте.

Учебный нейрон (т.е. такой нейрон, который используется в данных лабораторных работах) состоит из активации (U), смещения (S) активации, веса смещения (Ws) активации и из одной или нескольких связей, характеризующихся весом связи (W).

Рис. 1

Активация (U) рассчитывается в соответствии с формулой:

i – номер нейрона в котором рассчитывается активация;
n – номер нейрона имеющего связь с рассчитываемым нейроном i.

Из связанных определенным образом нейронов (узлов) строится нейронная сеть с определенным количеством входов и выходов. Обычно различают три типа узлов (нейронов):
1) входные (входной слой нейронов или Input layer),
2) выходные (выходной слой или Output layer),
3) скрытые слои нейронов (Hidden layers) или внутренние.

Рис. 2

И так, нейронная сеть - это система нейронов и связей между ними. Каждый нейрон обладает следующими характеристиками: U - активация нейрона, W - вес связи между нейронами, S - смещение активации нейрона, Ws - "смещение" для смещения активации нейрона.
О том, как работает такая нейронная, сеть будет написано чуть позже, а сейчас обратите внимание на то, как создать, используя язык С++, структуру описанную выше (Рис. 2).

Задание 4
Создайте функцию NeroNet(), в которой и будете работать с нейросетью. Для этого сразу после подключения классов, помогающих работать с нейросетью, запишите следующее:

void NeroNet()
{
    //тут будет код работы с нейросетью
}

Для создания нейросети вы должны создать объект класса NERO_NET (класс для работы с нейросетью). В процессе создания класса будет использован конструктор. Вот его прототип:

NERO_NET
(
    unsigned short int in_n,
    unsigned short int out_n,
    unsigned short int internal_c,
    unsigned short int internal_r
);

Рассмотрите прототип конструктора (с параметрами) класса CNero_Net:
in_n - Количество нейронов входного слоя,
out_n - Количество нейронов выходного слоя,
internal_c, internal_r - количество нейронов во внутреннем (скрытом) слое по вертикали и горизонтали соответсвенно.

Задание 5
Используя конструктор, описанный выше, создайте объект класса CNero_Net, где входных неронов 2, выходных 3, а внутренних 4 ( два ряда и в каждом ряду по 2 нерона) так, как это показано ниже. Данный код должен размещаться между операторных скобок функции NeroNet().

//создание нейросети (кол входных неронов, выходных, внутренних (число столбцов и строк))
NERO_NET* net = new NERO_NET(2,3,2,2);

И так сеть создана. Сейчас оцените то, что же у вас получилось?

Задание 6
Для этого воспользуйтесь методом toLog класса NERO_NET, единственный параметр которого, путь к файлу отчета, записав ниже приведенный код после создания объекта нейросети.

net->toLog("nero_net_log.txt");

И последний штрих! Не забывайте, что после создания и использования класса NERO_NET, необходимо освобождать память от объекта данного класса.

Задание 7
Освободите память от объекта класса NERO_NET, используя деструктор класса так, как показанно ниже:

//удаление нейросети

Нейросеть создается, удаляется, информации о ее состоянии выводиться в файл отчета. Осталось только активизировать функцию NeroNet().
