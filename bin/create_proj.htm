Создание окна

Лабораторная работа №1

СОЗДАНИЕ ОКНА ПРИЛОЖЕНИЯ

В этой лабораторной работе вы научитесь:

1. Создавать пустой проект в среде Visual C++ 6.0.
2. Записывать главную функцию main().
3. Создавать цикл обработки сообщений Windows.
4. Регистрировать и инициализировать пустое окно windows приложения.

1. Создание проекта

В Visual C++ для каждой новой программы создается проект. Проект - это информация о состоянии среды программирования и ничего более. В сущности в проект записывается настройки интерфейса, какие панели открыты, а какие закрыты и многое другое...
Сейчас вы будете учиться создавать новый проект, хотя предполагается, что вы уже должны это знать. Если это так, то освежите себе память. Если нет, то вперед учиться!

Задание 1
Для начала работы вам следует запустить Visual C++ 6.0. После нажать New на главной панели и в появившемся окне выбрать закладку Projects. На рисунке 1 вы можете видеть множество типов проектов. Нам же нужен Win32 Application. Выберите его.

Рис. 1

Задание 2
В поле Location введите путь к месту хранения вашего проекта, а в поле Project Name его имя. К примеру, Location: "D:\temp\work", а Project Name: "Direct3D". Кнопка Ok станет активна. Нажмите её, чтобы завершить выбор проекта.

Задание 3
Перед вами должно открыться окно Win32 Application - Step 1 of 1 (Рис. 2). Выберите An empty project (пустой проект). Нажмите Finish.

Рис. 2

Вот и все! Не правда ли легко?

После того, как создан совершенно пустой проект в него нужно добавить тот документ, в котором, собственно, будем писать.

Задание 4
Для этого намите опять New на главной панели и в появившемся окне выберите закладку Files.

На рисунке 3 вы можете видеть те файлы, которые можно создавать. Вам нужен C++ Source File.

Рис. 3

Задание 5
Выберите C++ Source File и в правой части окна в поле File name введите его название. К примеру, "D3DApplication". Нажмите Ok. Перед вами откроется белый лист.
